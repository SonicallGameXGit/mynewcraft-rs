## Installation

Don't panic, we're not in C++! :}}  
You're just need to have:  
 - git: https://git-scm.com/downloads  
 - Rust: https://www.rust-lang.org/tools/install

And Ctrl + C, Ctrl + V:
```bash
git clone https://github.com/SonicallGameXGit/mynewcraft-rs.git
```
```bash
cd mynewcraft-rs
```
```bash
cargo run --release
```
